Ciência de Dados




O que é Ciência de Dados?


Ciência de dados é um campo interdisciplinar que combina estatística, matemática, ciência da computação e conhecimento de domínio para extrair insights, conhecimentos e valor a partir de conjuntos de dados. O objetivo da ciência de dados é obter informações úteis e acionáveis a partir dos dados, a fim de tomar decisões informadas e resolver problemas complexos.
A ciência de dados envolve várias etapas, incluindo a coleta e armazenamento de dados, pré-processamento e limpeza dos dados, análise exploratória, modelagem estatística, construção de algoritmos de aprendizado de máquina e visualização dos resultados. Essas etapas são realizadas por cientistas de dados, que são profissionais especializados na aplicação de métodos e técnicas para extrair conhecimento dos dados.
A ciência de dados tem aplicações em diversas áreas, como marketing, finanças, saúde, ciências sociais, transporte, entre outras. Ela é usada para resolver problemas complexos, identificar padrões ocultos, prever tendências futuras, otimizar processos, tomar decisões baseadas em dados e impulsionar a inovação.
As habilidades necessárias para se tornar um cientista de dados incluem conhecimento em estatística, programação, aprendizado de máquina, visualização de dados e compreensão do domínio específico em que se está trabalhando. Além disso, a capacidade de comunicar efetivamente os resultados obtidos e tomar decisões baseadas em evidências são características importantes nessa área.


O que aprendemos?


Iniciamos a matéria com aulas de estatística, pois ela é fundamental na análise de dados. Em conjunto, aprendemos a realizar esses cálculos no python usando a plataforma google colab, as diversas bibliotecas existentes no python no auxiliaram a realizar os cálculos de forma rápida e eficiente. Exploramos conteúdos como amostragem (aleatória, estratificada e simplificada), aprendemos a realizar análises descritivas, como tendencia central (média, moda e mediana), fizemos o uso do recurso de plotagem de gráficos no python para estudar as distribuições e tendencias do dados.
Também aprendemos sobre escalas de medições e seus tipos. Logo em seguida entramos em medidas de dispersão (variância e desvio padrão). As medidas de dispersão nos ajudam a entender a distribuição dos dados e fornecem sua variabilidade. Ao plotar gráficos podemos ver a classificação e predição dos dados.
Aprendemos sobre distribuição probabilística para a compreensão do comportamento dos dados e suas diferentes distribuições (normal, binomial e Poisson). Estudamos diferentes tipos de funções e exploramos conceitos de cálculo, como os limites, derivadas e integrais.
